АЭРОБИКА



Аэробика - это групповые занятия под руководством тренера. Аэробика оказывает укрепляющее действие на сердечно-сосудистую и дыхательную системы, способствует снятию усталости, поддержанию формы и отличного настроения.
Разнообразие групповых программ позволит подобрать оптимальную нагрузку для максимальных результатов.

Виды аэробных тренировок:


АЭРОБНЫЕ и СМЕШАННЫЕ классы (50-55 мин)

ФИТНЕСС-КЛАСС - силовой урок аэробного характера на все крупные группы мышц с использованием различного оборудования для уменьшения жировой массы и развития общей силовой выносливости. Рекомендован для всех уровней подготовки.
ZUMBA - интенсивный урок аэробного характера с элементами латиноамериканских танцев - сальсы, меренге, реггетона, кумбии для уменьшения жировой массы и улучшения настроения. Рекомендован для всех уровней подготовки
STEP - интенсивный урок аэробного характера с использование степ-платформы для активной проработки мышц ног и ягодиц. Рекомендован для всех уровней подготовки.
STEP-INTERVAL - интенсивный урок: чередуются блоки аэробных и силовых упражнений на все крупные группы мышц с использованием степ-платформы и другого оборудования. Рекомендован для среднего уровня подготовки
POWERSTEP - интенсивный силовой урок аэробного характера на все крупные группы мышц с использованием степ-платформы и другого оборудования. Рекомендован для среднего уровня подготовки.
САЙКЛ - интенсивный аэробный урок на велотренажерах для активной проработки мышц ног и ягодиц, с использованием утяжелителей. Развивает координацию, силу и выносливость мышц. Рекомендован для среднего уровня подготовки.
AERODANCE - танцевальная аэробика. Высокая аэробная нагрузка, танцевальный ритм, хорошее настроение. Рекомендуется для любого уровня подготовки.

СИЛОВЫЕ КЛАССЫ (45-50 мин)

HOTIRON - силовой класс с использование мини штанг от 5 кг (вес подбирается индивидуально) для активной проработки проблемных зон. Рекомендован для всех уровней подготовки.
PUMPFITNESS - интенсивный силовой урок с использованием различных видов оборудования для коррекции фигуры в проблемных зонах. Рекомендован для среднего и высокого уровня подготовки.
SUPERBODY - силовой класс средней интенсивности направленная на проработку верхней части туловища (мышцы рук, спины, пресса). Рекомендован для всех уровней подготовки.
POWERBALL - урок смешанного характера не высокой интенсивности с использованием медбола, силовые упражнени¤ в сочетании с упражнениями на баланс и координацию. позволяет проработать все основные группы мышц в достаточно щадящем режиме. Рекомендован для всех уровней подготовки.
ABS - силовой класс: аэробная разминка + глубокая проработка мышц пресса и спины для всех уровней подготовки.
ФУНКЦИОНАЛЬНЫЙ ТРЕНИНГ - круговая тренировка дл¤ всех групп мышц, с использованием различных видов оборудования. Развивает координацию и выносливость мышц. Активное жиросжигание. Рекомендована для всех уровней подготовки.
КАЛЛАНЕТИКА - статические силовые упражнения с использованием собственного веса. Детальная проработка основных мышечных групп. Рекомендована для всех уровней подготовки.
FITBALL - силовой класс средней интенсивности c использованием большого мяча для всех уровней подготовки.
КРУГОВАЯ ТРЕНИРОВКА - интенсивный силовой класс: упражнения на тренажерах и со свободными весами на все крупные группы мышц выполняются по кругу. Рекомендован для всех уровней подготовки.
POWER - CLASS - силовой класс для всех уровней подготовки, направленный на тренировку всех групп мышц, с использованием различных видов оборудования.
POWERSTEP - урок смешанного формата, в котором упражнения на степ-платформе чередуются с силовыми упражнениями. Тренирует силовую выносливость. Рекомендован для подготовленных.
SUPERSCULPT - силовая тренировка, направленная на мышцы ног, ягодиц и мышцы пресса. Проходит с использованием дополнительного оборудования. Подходит для всех уровней подготовки.

РАЗУМНОЕ ТЕЛО (50-55 мин)

PILATES - урок на детальную проработку глубоких мышечных слоев, что позволяет избавиться от лишнего веса, улучшить гибкость, избавиться от болей в спине. Рекомендован для всех уровней подготовки.
PILATES+ КАЛЛАНЕТИКА - урок из 2 блоков: силовая тренировка с доп. оборудованием (мышцы ягодиц, груди, спины, живота) + упражнения на релаксацию и восстановление. Рекомедован для всех уровней подготовки.
PILATES - BALL - урок на укрепление основных групп мышц с использованием фитбола (бол. мяч) в сочетании с упражнениями на координацию движений и растяжку. Рекомендован для всех уровней подготовки.
